## ‚ö° Getting Started with Obsidian

1. **Open Vault**  
   - Launch Obsidian and select `Open folder as vault`.
   - Choose the `PFE Vault/` folder as your vault.

2. **Navigation & Linking**  
   - Use Obsidian‚Äôs **backlinks** and **graph view** to see how notes connect.  
   - Internal links (`[[Note Name]]`) let you jump instantly between related topics.

3. **Tagging System**  
   - Tags (e.g., `#gameplay`, `#AI`, `#tutorial`) help categorize content across folders.  
   - Benefits:
     - Quickly find related notes regardless of folder location.
     - Filter your notes by type or theme in search and graph view.

4. **Search & Metadata**  
   - Use Obsidian‚Äôs search to find players, mechanics, or code references instantly.  
   - Optional: add YAML frontmatter to notes for metadata like `status: draft` or `priority: high`.
