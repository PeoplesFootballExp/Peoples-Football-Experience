
## 1. Install Godot

1. Download Godot 4.x (latest stable version) from [**Godot's Website**](https://www.godotengine.org)
> Install the normal version NOT the .NET version

2. Place Godot somewhere easy to access such as Desktop. Launch it and import ../Peoples-Football-Experience/godot/people's-football-experience as a new project.

## 2. Install Rust

1. Install Rust from the instructions from the [Official Rust Website](https://rust-lang.org/tools/install/)
2. Ensure Rust is up to date:
	`rustup update`
3. Verify Installation:
	`rustc --version
	`cargo --version`
