
# Contributing to People's Football Experience

Thank you for your interest in contributing. People's Football Experience is a free, inclusive, and community-driven football game. Your contributions help grow a vibrant ecosystem of open-source football experiences.

## Code Contributions

### Supported Languages

- **Rust**: Follow Rust's official style guidelines for formatting, naming, and idioms. For the most part, `rustfmt` will take care of most formatting.
- **GDScript**: Follow GDScript style recommendations for code formatting, signals, and naming.

### Guidelines
- Write clean, readable, and maintainable code
- Include comments for complex or non-obvious logic
- Add tests when possible
- Keep commits small, atomic, and descriptive
- Link pull requests to related issues when applicable
