
# Identifying Information

## Fields

- id
- name
- official_name
- alt_name
- code
- ISO-3 codes

P1448 - Official Name in English only 
P1549 - Demonym in English only 
P85 - National Anthem audio 
P17 - Country (mainly for dependent territories, if not present then null) 
P3896 - Geoshape (if any) for possible map in game 
P625 - Coords (if any) for possible map in game 
P1082 - Population with point in time data (for filtering later in code) as seperate column 
P2131 - Nominal GDP with point in time data as seperate column 
P2936 - Language Used with qualifies like number of speakers and has characteristic as seperate columns 
P2046 - Area with any qualifiers present as seperate columns 
P41 - Flag image with start and end time qualifiers as seperate columns 
P298 - ISo 3166-1 alpha-3 code
